# ER-TCNet

This repository contains research code for **ER-TCNet**, an experimental method built around 3D Gaussian reconstruction and triplane-based neural representation.

The codebase includes model components, dataset utilities, training/evaluation scripts, and a small set of demo assets. It is intended primarily for internal research reference and result tracking.

## Repository Layout

- `scene/` - core network modules and Gaussian prediction components
- `gaussian_renderer/` - rendering interface
- `datasets/` - dataset loading and camera utilities
- `configs/` - experiment configuration files
- `demo_examples/` - lightweight visual examples
- `train_network.py`, `eval.py` - main research scripts

## Notes

This is a research snapshot. Some data, checkpoints, generated outputs, and local experiment logs are intentionally not included in the repository.
